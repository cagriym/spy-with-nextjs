# Spy with Next.js

Next.js tabanli bir arayuz/prototip calismasi. Radix UI bilesenleri ve Tailwind uzerinden hizli UI denemeleri yapilir.

## Teknolojiler
- Next.js 15 (App Router)
- React 19
- Radix UI
- TailwindCSS

## Kurulum
```bash
npm install
```

## Gelistirme
```bash
npm run dev
```

## Build
```bash
npm run build
npm run start
```
